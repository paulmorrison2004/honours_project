# honours_project

## Overview

This repository contains the code for my BSc Honours thesis on predicting climate vulnerability in vertebrate guilds of the Madrean Sky Islands using a hybrid machine learning framework.

The project combined occurrence, climatic, elevation, and physiological trait data to model habitat suitability under future warming scenarios. I used XGBoost alongside mechanistic thermal constraints to improve biological realism and examine how thermal limits influence projected habitat loss, range shifts, and extinction risk.

The work was developed as a scalable and reproducible pipeline for trait-based ecological forecasting, with the broader aim of improving how climate vulnerability is identified across functionally distinct groups.

This repository is included primarily to demonstrate the analytical workflow behind the thesis and my experience applying machine learning to ecological and conservation questions.
